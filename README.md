# DRONE
基于c8t6 和rtos的无人机，遥控器和飞控板两个适配板，typec和锂电池充放电，2.4g通信，摇杆和按键操纵，数模地分离，可以稳定飞行，有待调试
Drone based on C8T6 and RTOS, with two adapter boards for the remote controller and flight control board, Type-C and lithium battery charging and discharging, 2.4G communication, controlled via joystick and buttons, digital and analog separation, capable of stable flight, pending debugging
